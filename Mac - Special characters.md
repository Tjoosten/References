# Reference - Mac special characters

This reference is writed for a macbook 13' inch.

| Input                    | Output                        |
| ------------------------ | ----------------------------- |
| `shift` + `alt` + `L`    | output = horizontal stripe    |
| `shift` + `alt` + `(`    | output = `[`                  |
| `shift` + `alt` + `)`    | output = `]`                  |
| `alt` + `(`              | output = `{`                  |
| `alt` + `)`              | output = `}`                  | 
| `shift` + `alt` + `$`    | output = `¥`                  |
| `shift` + `alt` + `£`    | output = `#`                  |
| `shift` + `alt` + `:`    | output = `\`                  |
| `shift` + `alt` + `§`    | output = `å`                  |
| `shift` + `alt` + `è`    | output = `»`                  |
| `shift` + `alt` + `&`    | output = `´`                  |
| `shift` + `alt` + `é`    | output = `„`                  |
| `shift` + `alt` + `"`    | output = `”`                  |
