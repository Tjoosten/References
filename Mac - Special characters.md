# Reference - Mac special characters

This reference is writed for a macbook 13' inch.

| Input                    | Output                        |
| ------------------------ | ----------------------------- |
| `shift` + `alt` + `L`    | output = horizontal stripe    |
| `shift` + `alt` + `(`    | output = `[`                  |
| `shift` + `alt` + `)`    | output = `]`                  |
| `alt` + `(`              | output = `{`                  |
| `alt` + `)`              | output = `}`                  | 
| `shift` + `alt` + `$`    | output = `¥`                  |
| `shift` + `alt` + `£`    | output = `#`                  |
| `shift` + `alt` + `:`    | output = `\`                  |
| `shift` + `alt` + `§`    | output = `å`                  |
| `shift` + `alt` + `è`    | output = `»`                  |
| `shift` + `alt` + `1`    | output = `´`                  |
| `shift` + `alt` + `2`    | output = `„`                  |
| `shift` + `alt` + `3`    | output = `”`                  |
| `shift` + `alt` + `4`    | output = `’`                  |
| `shift` + `alt` + `8`    | output = `Û`                  |
| `shift` + `alt` + `9`    | output = `Á`                  |
